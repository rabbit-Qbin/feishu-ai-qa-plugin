# 飞书选品打分与分类规则（精简版）

## 1. 使用场景

- 数据来源：卖家精灵「选产品」功能导出的候选产品列表。
- 目标：在大量候选 ASIN 中，基于数据进行优中择优，而不是只看感觉。
- 所有分数都在 0–100 之间，数值越高代表越优（注意：竞争强度为**反向得分**，分高代表竞争越低、越好）。

---

## 2. 三大核心维度

### 2.1 需求趋势得分

- 综合衡量产品的**销量规模与增长潜力**，由 9 个子项组成：
  - 月销量_得分（权重 20）
  - 月销售额_得分（18）
  - 月销量增长率_得分（13）
  - 小类BSR_得分（11）
  - 大类BSR增长率_得分（9）
  - 大类BSR_得分（7）
  - 评分数_得分（8）
  - 月新增评分数_得分（7）
  - 评论增长活跃度_得分（7）
- 子项通过与「指标基准表」中的分位值（min/q25/median/q75/max）分段插值计算，最终加总为 **0–100 的需求趋势得分**。

### 2.2 竞争强度得分（反向）

- 衡量**竞争压力大小**，由 6 个子项组成：
  - 卖家数_得分（22）
  - 变体数_得分（15）
  - 评分数_反向_得分（15）
  - LQS_反向_得分（12）
  - Listing质量综合_反向_得分（10）
  - 上架天数_得分（26）
- 这些是**反向指标**：卖家越少、变体越少、评分越少、LQS 越低、Listing 越差、上架越短 → 竞争空间越大 → 得分越高。
- 加总得到 **0–100 的竞争强度得分**，数值越大代表竞争越低、机会越好。

### 2.3 利润空间得分

- 衡量产品的**毛利与费用结构**，由 4 个子项组成：
  - 毛利率_得分（40）
  - 包装尺寸分段_得分（30）：小号标准尺寸得分最高，尺寸越大得分越低。
  - FBA费用占比_得分（25）：FBA占比越低得分越高。
  - 净利率估算_得分（5）
- 加总得到 **0–100 的利润空间得分**。

### 2.4 综合得分

- 综合考虑三大维度：
  - 综合得分 = 需求趋势得分 * 0.35 + 竞争强度得分 * 0.35 + 利润空间得分 * 0.30
- 综合得分越高，说明产品在需求、竞争、利润三方面都更均衡、更有优势。

---

## 3. 初步产品分类（四象限）

- 使用「指标基准表」中的两个中位数：
  - `median_需求趋势得分` → 需求中轴（X 轴）
  - `median_竞争强度得分` → 竞争中轴（Y 轴）
- 按需求趋势得分（横轴）与竞争强度得分（纵轴）划分为四类：

1. **畅销爆品**  
   - 条件：需求趋势得分 ≥ 需求中轴 且 竞争强度得分 ≥ 竞争中轴  
   - 含义：需求旺盛、竞争相对可控，是当前类目中的明星产品。

2. **稳健产品**  
   - 条件：需求趋势得分 ≥ 需求中轴 且 竞争强度得分 < 竞争中轴  
   - 含义：需求不错但竞争偏激烈，更适合已有资源/老卖家稳健跟进。

3. **潜力产品**  
   - 条件：需求趋势得分 < 需求中轴 且 竞争强度得分 ≥ 竞争中轴  
   - 含义：目前需求一般但竞争较低，属于潜力蓝海方向。

4. **淘汰产品**  
   - 其他情况  
   - 含义：需求与竞争都不占优，优先考虑淘汰或暂不进入。

---

## 4. 选品结论与优先级

### 4.1 选品结论（基于综合得分分位）

- 结合综合得分与「指标基准表」中的分位值，将产品分为三档：
- 指标基准表中关于综合得分的关键字段：
  - `综合得分_q25`：所有产品综合得分的 25% 分位（Q25，下四分位）
  - `综合得分_median`：所有产品综合得分的中位数
  - `综合得分_q75`：所有产品综合得分的 75% 分位（Q75，上四分位）

1. **可选**  
   - 条件：综合得分 ≥ `综合得分_q75`  
   - 含义：综合得分处于样本的前 25%，属于当前批次产品里的优等生，建议重点关注，可以进入候选清单。

2. **谨慎选择**  
   - 条件：`综合得分_q25` ≤ 综合得分 < `综合得分_q75`  
   - 含义：综合水平在中间区间，既有机会也有不足，需要结合自身资源与供应链进一步评估。

3. **不推荐**  
   - 条件：综合得分 < `综合得分_q25`  
   - 含义：综合得分处于样本的底部区间，当前数据维度下不具备明显优势，优先排除。

### 4.2 优先级

- 在选品结论基础上，再细分运营优先顺序：

1. **高优先级**  
   - 条件：综合得分 ≥ `综合得分_q75`  
   - 对应结论多为「可选」，是当前批次中最值得优先推进的一档。

2. **中高优先级**  
   - 条件：`综合得分_median` ≤ 综合得分 < `综合得分_q75`  
   - 适合作为第二梯队关注对象，结合运营资源选择性进入。

3. **中优先级**  
   - 条件：`综合得分_q25` ≤ 综合得分 < `综合得分_median`  
   - 综合表现中等，更适合作为观察和储备项目。

4. **中低优先级 / 低优先级**  
   - 条件：综合得分 < `综合得分_q25` 或选品结论为「不推荐」  
   - 当前阶段不建议投入较多资源，只作备案或放弃。

> 小结：所有高/中/低阈值都不是写死的具体分数，而是基于当前批次产品的综合得分分布（Q25 / 中位数 / Q75）动态计算，能随类目与样本变化自动调整。

---

## 5. 最终产品分类

在「初步产品分类」和「综合得分」基础上，形成更易理解的最终标签：

- **蓝海爆品**  
  - 条件：初步产品分类为「畅销爆品」且综合得分 ≥ `综合得分_q75`  
  - 含义：需求旺盛、竞争可控、利润合理，是当前批次中最优先主推方向。

- **潜力蓝海**  
  - 条件：初步产品分类为「畅销爆品」，且综合得分在 `综合得分_q25` 与 `综合得分_q75` 之间  
  - 含义：目前已有一定表现，但仍有提升空间，通过优化 Listing / 供应链有望向蓝海爆品跃迁。

- **稳健产品**  
  - 条件：初步产品分类为「稳健产品」且综合得分 ≥ `综合得分_q75`  
  - 含义：适合已有资源、希望在该赛道稳健扩充品线的卖家。

- **观察产品**  
  - 其他尚有一定分数的产品  
  - 含义：可以放入观察池，结合趋势和对手变化再做决策。

- **暂不考虑**  
  - 条件：初步分类为「淘汰产品」或综合得分明显偏低  
  - 含义：当前阶段不建议投入资源。

---

## 6. 使用建议（对 AI）

- 回答用户时，请尽量：
  - 用「需求趋势 / 竞争强度 / 利润空间」三维度解释原因；
  - 结合「初步产品分类 + 最终产品分类 + 优先级」给出结论；
  - 说明这是在「卖家精灵选产品导出的全量候选产品」中做的优中择优判断，而非保证成功的唯一标准。
